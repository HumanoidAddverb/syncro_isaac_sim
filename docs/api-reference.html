API Reference

Complete reference for the WebSocket protocol, Python client library, CLI arguments, and configuration files.

WebSocket Protocol

The NVIDIA Isaac Sim server exposes a WebSocket API on port 8765. All messages are JSON-encoded UTF-8 text frames.

set_joints

Set target joint positions (radians). Only listed joints are updated.

// → Client sends:
{
  "cmd": "set_joints",
  "positions": {
    "joint1": 0.0, "joint2": -0.785, "joint3": 1.571,
    "joint4": 0.0, "joint5": 0.0, "joint6": 0.0
  }
}
// ← No response (fire-and-forget)
ℹ️ Note
The command queue has maxsize 1. Stale commands are dropped so PhysX always applies the latest target.

get_state

Query current joint positions and velocities from Isaac Sim.

// → {"cmd": "get_state"}
// ← {"joint_positions": {...}, "joint_velocities": {...}}

set_gripper_state

Set gripper to "open" or "closed". Applies mimic multipliers automatically via PhysX.

{"cmd": "set_gripper_state", "gripper_state": "open"}
{"cmd": "set_gripper_state", "gripper_state": "closed"}
Statefinger_jointOuter knucklesInner fingers
open0.0 rad0.0 rad0.0 rad
closed0.8 rad0.8 rad−0.8 rad

get_gripper_state

// → {"cmd": "get_gripper_state"}
// ← {"gripper_state": "open"}

SyncroSimClient (Python)

Thread-safe Python client in syncro_sim_client.py. Raw socket WebSocket, no external deps.

Constructor

client = SyncroSimClient(robot="syncro5", host="localhost", port=8765)

connect() / disconnect()

client.connect(timeout=10.0)
client.disconnect()
# Or context manager:
with SyncroSimClient(robot="syncro5") as client:
    state = client.get_state()

set_joint_positions(positions)

client.set_joint_positions({"joint1": 0.0, "joint2": -0.785})

set_joint_positions_degrees(positions_deg)

client.set_joint_positions_degrees({"joint2": -45.0, "joint3": 90.0})

set_all_joints(values, degrees=False)

client.set_all_joints([0.0, -0.785, 1.571, 0.0, 0.0, 0.0])
client.set_all_joints([0, -45, 90, 0, 0, 0], degrees=True)

get_state()

state = client.get_state()
print(state["joint_positions"])
print(state["joint_velocities"])

mirror_real_robot(obs, degrees=True)

Translate Addverb real-robot observation (LeRobot format) to Isaac Sim joint targets.

client.mirror_real_robot({
    "shoulder_pan.pos": -6.36, "shoulder_lift.pos": -51.95,
    "elbow_flex.pos": 16.73, "wrist_flex.pos": 89.25,
    "wrist_roll.pos": -51.85, "gripper.pos": 0.0,
})
💡 Tip
Real → sim mapping: shoulder_pan → joint1, shoulder_lift → joint2, etc. Unknown keys ignored.
